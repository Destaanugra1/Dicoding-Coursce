Apa itu SQL Injection?

SQL Injection adalah teknik serangan siber di mana penyerang menyisipkan kode SQL berbahaya ke dalam input aplikasi web yang kemudian dieksekusi oleh database server.

Serangan ini terjadi ketika aplikasi web tidak memvalidasi atau membersihkan input pengguna dengan tepat sebelum menyisipkannya ke dalam query SQL. Penyerang dapat memanipulasi query untuk mengakses, memodifikasi, atau menghapus data sensitif dalam database.

Contoh serangan sederhana adalah memasukkan ' OR '1'='1 pada kolom username login, yang dapat mengakali sistem untuk memberikan akses tanpa password yang benar. Untuk mencegah SQL Injection, pengembang harus menggunakan prepared statements, parameterized queries, dan validasi input yang ketat.
